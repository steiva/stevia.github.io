排序总结

包括一些概念加深理解

一些概念准备（之前没有系统研究过）

  1. 面向对象
  • 万物皆对象
  • 程序时对象的集合，通过发送消息告知彼此所要做的
  • 每个对象都有自己的有其他对象构成的存储
  • 每个对象都有其类型
  • 某一个特定类型的所有对象都可以接受同样的消息
  • 状态、行为、标识
  1. 接口
  • 就扣确定了对某一特定对象所发出的请求。
    eg：light it= new Light();
    li.on();
    （这里面，on（）就是一种接口）
  1. 边界
  • public:后面的元素任何人可用
  • private：内部方法可以用
  • protected：继承类可以访问protected不能访问private
  1. 继承
  • 基类和导出类有相同的基础接口
  • extends
  1. 容器
  • c++ STL
  • java 一些类库 List Map Set
  • 不同的容器提供不同的接口和外部行为
  • 不同容器对于某些操作具有不同的效率
  • 参数化类型 在java中成为泛型 用一对尖括号表示
  1. 对象创建和生命周期
  • java 采用动态内存分配方式 eg： new
  1. 异常处理：处理错误
  • “抛出”、“捕获”
  • java 内置了异常处理
  1. 并发编程
  • 彼此独立运行的部分称之为线程
  • 隐患 共享资源 eg：两个进程不能同时向打印机发送信息
  1. java与internet
  • 客服/服务器 CGI（通用网关接口）
  • 插件 plug-in
  • 脚本语言
  • 服务端 数据库搜索请求
  1. 引用操纵对象
  • String s ;String s=”asdf” ;String s= new String(“asdf”);
  • new 就是用来和一个新的对象关联
  • 寄存器、堆栈、堆、常量存储、非RAM存储
  • 特例 常量 boolean char int….
  • 数值 c/c++中数组就是内存块 比如要一开始要定义数组的大小
  • 内部有一个“垃圾回收器”，辨别不会在被引用的对象
  1. 创建新的数据类型：类
  • class ATypeName; ATypeName a=new AtypeName
  1. 方法、参数、返回值
  • objectName.methodName(arg1,arg2,arg3)
    eg: int x=a.f();
  • import import java.util.ArrayList
  • static 引用方法有两种：1. 通过对象去定位它 2. 直接通过类名引用

java自学笔记（路线）

2020-09-01
  1. 优势 相关环境 很健全 还有开源项目
  2. 和很多其它的语言相比 应用比较广

基础知识

  • 编程语言 java 找开源项目 自己动手上手（已经学过c++和python 上手相对简单）
    （java编程思想）
  • 基本算法与数据结构（leetcode刷题）
  • 基本网络知识： TCP/IP HTTP/HTTPS
  • 基本的设计模式

（注意配上项目动手，基础打牢）

工具方面

  • 操作系统 Linux（Centos，Ubuntu…）
  • 代码管理（git/svn）
  • 持续集成(CI/CD):jenkins
  • java项目管理工具： Maven/ Gradle

框架

应用层框架

  • ssh spring + struct + hibernate~
  • ssm: spring springmvc mybatis
  • spring boot (很好上手，自动配置)

各种中间件

  • MQ(message queue)消息队列
  • RPC 通信框架 gRPC thrift dubbo springcloud
  • elasticsearch 数据库 搜索引擎

(还有很多，根据项目情况选用)

数据库

  • SQL mysql/ postgre SQL
  • NoSQL: Redis Memcached mongodb elasticsearch

架构

分布式 微服务架构

  • spring cloud
  • dubbo
  • RPC通信框架

虚拟化/容器化基础

  • Docker 容器化（搭建集群啥的、黑盒、简单、有点像虚拟机）
  • k8s kubernetes

关注源码/性能（学有余力）

  • JDK 的源码及部分设计思想
  • Spring源码 (ioc容器)
  • JVM 细节和排错
  • 高并发/高可用

Java相关书籍

  • Java编程思想（当工具书用吧）

  • Java并发编程实战

  • 深入理解Java虚拟机 （jvm 面试有人提问）

  • 函数式编程思想

  • TCP/IP 详解（必须看）

  • 鸟哥的linux私房菜

  • Spring MVC MyBatis开发 从入门到项目实战 （入门使用 后面可以先看看spring boot spring cloud）

  • Spring 技术内幕 （Ioc Aop 实现原理等等 能不能面向切面编程的范式 看的比较难 最好先有自己动手的经验之后 再抽空看看）

  • Elasticsearch 可以考虑看看

  • Redis 入门指南 （快餐式的。挺好入门 然后可以看官方文档）
